To run this project 
compile the file:

--> gcc main.c Robby.c Corey.c Kyle.c -lpthread

and exectute the program with 6 int arguments:

--> ./a.out 2 10 5 3 10 1000 

For the back up file with all the functions in one file instead of separate:

--> gcc backup.c -lpthread

and exectute the program with 6 int arguments:

--> ./a.out 2 10 5 3 10 1000
